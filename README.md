# MkEdu

MkEdu is a cross-platform app designed for schools (universities, polytechnics, etc.), connecting students and faculty via real-time group chats, lecture recording and uploads, and a moderated educational marketplace. The app solves missed lectures, provides key resources 24/7, and supports interaction between students and lecturers across hierarchical school structures.

## MVP Features
- Email authentication and role-based signup (Student/Lecturer)
- Group chat for School, Faculty, Department, Course
- Local audio recording with offline queue and automatic upload
- Marketplace: students can list items for sale (lecturer approval needed)
- Lecturer rosters, remarks, and PDF export
- Push notifications for messages and uploads

## Stack
- Flutter (iOS/Android/Web/Desktop)
- Supabase (Auth, Realtime, Storage, Database)

## Setup
1. Clone the repo: `git clone git@github.com:Mkmakama13/MkEdu.git`
2. Install Flutter & dependencies
3. Connect your Supabase project
4. Run locally: `flutter run`
